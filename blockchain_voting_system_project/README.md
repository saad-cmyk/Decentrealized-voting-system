# Blockchain Smart Contract Based E-Voting System

A decentralized voting system built using:

- HTML, CSS, JavaScript
- Web3.js
- Ethereum Smart Contract
- Express.js backend

## Features

- MetaMask wallet connection
- Smart contract interaction
- Secure vote casting
- Real-time vote results display

## Installation

npm install

## Run

npm start

Open in browser:
http://localhost:3000

## Requirements

- MetaMask browser extension
- Deployed Ethereum smart contract
